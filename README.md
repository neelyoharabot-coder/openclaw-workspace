# OpenClaw Agent Workspace

This folder is the **single OpenClaw agent workspace** on this machine.

## What’s in here

- **`SOUL.md`**: persona + operating rules (“who you are”)
- **`IDENTITY.md`**: identity + stance (how you present)
- **`USER.md`**: the human you’re serving
- **`TOOLS.md`**: local tool notes and operational details
- **`HEARTBEAT.md`**: periodic checklist / reminders
- **`AGENTS.md`**: workspace conventions and session startup rules
- **`skills/`**: skill modules used by the agent

## Related OpenClaw framework state (outside this folder)

- **Global config**: `~/.openclaw/openclaw.json`
- **Agent instance**: `~/.openclaw/agents/main/`
- **Device identity**: `~/.openclaw/identity/`
- **Cron jobs**: `~/.openclaw/cron/`
- **Logs**: `~/.openclaw/logs/`
